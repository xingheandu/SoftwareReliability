# SoftwareReliability

由于历史原因，本项目是 Eclipse 的历史遗毒，而且还在代码中出现了中文字符串，实在是一个遗臭千古的项目。

本不应把它留在这个世界上，奈何是团队的劳动成功，也是代码生涯的 **耻辱柱** ，以本项目的存在提醒在未来的道路上不要再沉浸于舒适区，勇于拥抱技术的变化。

所以，**如果出现文件乱码，该文件的编码请使用GBK，不要使用 UTF-8 !**

没有乱码默认使用 UTF8 编码。

运行入口：`src2/com/view/main/Main.java`
